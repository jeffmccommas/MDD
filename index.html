design.

Design isn’t only about creating beautiful aesthetics. Commonly overlooked is that web developement is problem solving. Because design is a problem solving activity, you can measure how good a design is by how well it solves a particular problem. When I am hired, my role is to come up with clean code that solves the problem. I am a developer who matches up well with my clients because if have a passion for the content and can think at a high design level.
