HTML5 Bootstrap Responsive Design

WEB DESIGN BUILT WITH HTML5 BOILERPLATE

Aronis Builders is a well established General Contractor located on Boston's North Shore. They came to me asking for a simple but effective web design that there customers and clients could view on the smartphones and tablets as well as full screen desk tops. The solution was a Fully Responsive HTML5 web design

  • Built with the HTML5 Boilerplate Framework using Bootstrap responsive framework. This site is fully responsive to all screen sizes and media queries.
  • Next I integrated approximately 13 different Jquery library plug-ins to give the site some nice animation and user interactivity.
  • Because of so many Java-script HTTP requests I used UglifyJS Java-script minification to minify the files into just one file.
